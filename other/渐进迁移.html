# 渐进迁移

Uma 提供了中间件，方便 Koa 或者 Koa 衍生框架采用渐进迁移的方式迁移至 Umajs 框架。

# 中间件

# 使用

Uma 提供了 middleware(options: TUmaOption, app: Koa) 静态方法返回中间件供使用，示例如下：

# 方式1

const app = new Koa();

Uma.middleware(options, app).then((mw) => {
    app.use(mw);

    app.listen(8058);
})

# 方式2

(async () => {
    const app = new Koa();

    app.use(await Uma.middleware(options, app));

    app.listen(8058);
